Everything you need for JS Monitoring

Equipped with cutting-edge tool to intercept js traffic and unleash AI on potential vulnerabilities.

Target Domain Filtering

Monitor specific domains with wildcard support (*.target.*, *.target.com) to cut through the noise.

Multi-Model AI Scanning

Analyze code via Google Gemini, OpenAI ChatGPT, Anthropic Claude, or DeepSeek for vulnerabilities.

Real-time Interception

Capture all JS/JSX/TS/TSX/JSON requests on the fly with live status monitoring.

Advanced Regex Search

Search by domain, path, or URL with full regex support, inversions, and favorites filtering.

Robust Export Options

Export captured request data as structured JSON or pure URL lists for your external scripts.

Secure & Local

100% Local Storage. API keys are strictly kept in your browser. No tracking, ever.

Installation

Manual Install

  1. Download the latest version from releases
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable Developer Mode (toggle in top-right)
  4. Click Load unpacked
  5. Select the extracted extension folder
  6. Pin the extension for easy access

Chrome Web Store release is upcoming. For now, please use the manual installation method above.

Quick Start Workflow

1

Configure Target

Click the extension icon and add target patterns like *.target.com.

2

Start Monitoring

Click START_MONITORING. The badge will turn green and requests will populate.

3

Filter & Review

Use regex .* or invert i to find specific code chunks.

4

AI Scan

Hit SCAN on any URL to analyze it against your chosen AI model.
